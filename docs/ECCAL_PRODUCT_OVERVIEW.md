# 報數據 (ECCAL) - 產品功能完整介紹

## 📌 產品定位

**報數據 (ECCAL)** 是一個專為電商廣告主打造的**智能廣告分析與預算規劃平台**，整合 Facebook Marketing API、Google Analytics 4 和 AI 技術，提供從診斷、規劃到執行的完整廣告管理解決方案。

**核心價值**：
- 🎯 **精準預算規劃** - 科學化的預算分配演算法
- 🤖 **AI 智能診斷** - GPT-4 驅動的廣告健檢與優化建議
- 📊 **數據驅動決策** - 整合真實 Facebook 與 GA4 數據
- 🌐 **多語言支援** - 繁中、英文、日文三語系
- 🔐 **跨平台整合** - SSO 單一登入，串接多個子服務

### 2️⃣ **廣告預算計算機 (Ad Budget Calculator)**

**Google Analytics 4 整合的智能預算計算工具**

#### **功能特色**

**📈 GA4 自動數據導入**
- OAuth 授權串接 Google Analytics 4
- 自動抓取電商關鍵指標：
  - 平均訂單價值 (AOV)
  - 轉換率 (Conversion Rate)
  - 交易數據
- 多幣別支援（TWD / USD / JPY）

**🧮 智能預算計算**
- **目標導向** - 輸入目標營收自動計算所需預算
- **精準流量估算** - 基於轉換率計算所需流量
- **多幣別 CPC** - 各市場差異化點擊成本
- **ROAS 預測** - 自動計算預期投資報酬率

**💾 PDCA 計畫管理**
- 儲存計算結果
- 命名與分類專案
- 歷史計畫檢視
- 設定活躍計畫

**📊 結果呈現**
- 視覺化圖表
- 詳細數據表格
- 可匯出報告
- 移動裝置友善介面

#### **計算公式**

```
所需流量 = (目標營收 ÷ AOV) ÷ 轉換率
廣告預算 = 所需流量 × CPC
預期訂單數 = 所需流量 × 轉換率
預期 ROAS = 目標營收 ÷ 廣告預算
```

**適用場景**：
- ✅ 新產品上市預算規劃
- ✅ 促銷活動預算評估
- ✅ 年度廣告預算編列
- ✅ 投資報酬率預測

### 3️⃣ **活動預算規劃器 (Campaign Budget Planner)** 🚀

**智能活動預算分配系統（進階功能）**

#### **功能特色**

**🎯 多階段預算分配**

支援 **3-60 天活動規劃**，智能分配至以下階段：

| 階段 | 天數比例 | 預算比例 | 目的 |
|------|---------|---------|------|
| **預熱期** | 固定 4 天 | 4% | 建立期待感 |
| **起跑期** | 彈性 | 45% | 搶先衝刺 |
| **活動期** | 彈性 | 30% | 維持熱度 |
| **倒數期** | 固定 3 天 | 25% | 最後衝刺 |
| **回購期** | 固定 7 天 | 15% | 再次購買 |

**📅 智能期程規劃**
- **3 天短促** - 簡化為三階段 (50% / 25% / 25%)
- **4-9 天中期** - 起跑 / 活動 / 倒數三階段
- **10-60 天長期** - 完整五階段規劃

**📊 每日預算細分**
- 每日預算明細
- 每日流量目標
- 每日預期訂單
- 每日預期營收

**🎨 視覺化呈現**
- 預算分配圓餅圖
- 每日預算趨勢圖
- 階段性數據表格
- 可匯出 Excel/PDF

**🔒 會員權限控管**
- **免費會員** - 3 次試用
- **Pro 會員** - 無限次使用
- 後端計算防篡改
- 使用次數即時追蹤

#### **演算法特點**

```javascript
// 智能流量分配演算法
階段流量 = 總流量 × 階段百分比 × 強度係數
每日流量 = 階段流量 ÷ 階段天數

// 預算計算
每日預算 = 每日流量 × CPC
每日預期訂單 = 每日流量 × 轉換率
每日預期營收 = 每日預期訂單 × AOV
```
